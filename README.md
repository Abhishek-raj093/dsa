# dsa

This folder contains a large collection of Java source files solving algorithm and data structure problems. Most files are standalone Java classes or solutions for coding challenges, including array manipulation, graph traversal, dynamic programming, string processing, and more.

## Contents

- Java source files (`*.java`) for individual problems
- Some compiled class files (`*.class`)
- Supporting utility classes such as `FenwickTree.java`, `SegmentTree.java`, and others

## Usage

To compile a Java file:

```bash
javac <filename>.java
```

To run a compiled class:

```bash
java <MainClassName>
```

If the file contains a `main` method, you can execute it directly after compilation.

## Notes

- Many files appear to solve competitive programming or LeetCode-style problems.
- Filenames are often the problem name or a descriptive title.
- Some files may depend on custom input or test scaffolding not included in this folder.

## Suggested Workflow

1. Pick a Java file to review or test.
2. Compile it with `javac`.
3. Run it with `java` if it contains a `main` method.
4. Add tests or a driver class if needed to validate specific problem cases.
